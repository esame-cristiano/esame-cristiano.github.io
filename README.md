# 🌐 Esame terza media cristiano

## 📋 Descrizione Progetto

Questo è un sito web educativo creato per un esame di terza media. Il progetto affronta importanti temi legati al mondo digitale:

- **Il Digitale**: Introduzione base e semplice primo argomento e come usare internet in modo sostenibile
- **Cybersecurity**: Come proteggere se stessi online
- **Cyberbullismo**: Conseguenze e Metodi per batterlo

- **HTML5**: Struttura semantica

- ✅ Nessun dato personale raccolto

## 🐛 Troubleshooting

### Il sito non carica
- Verifica che tutti i file (HTML, CSS, JS) siano nella stessa cartella
- Prova con un browser diverso
- Svuota la cache del browser

### Le animazioni sono lente
- Riduci le schede aperte del browser
- Prova con un browser diverso
- Disabilita le estensioni del browser
